# XMindCase

## 安装
```
pip install git+https://github.com/YeKelvin/xmindcase.git
```

## 开发
```
git clone
cd xmindcase
rye sync
```

## XMind 用例格式
详情请参考 [testcase.xmind](https://github.com/YeKelvin/xmindcase/blob/master/testcase.example.xmind)

## 使用说明
```bash
xmindcase excel '文件路径' [--sheet 'sheet名称'] [--debug] [--help]
```
